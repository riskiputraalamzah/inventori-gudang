# Best Practice Proyek

Folder ini menyimpan keputusan kerja sebelum dan selama pengembangan.

## File utama

| File | Fungsi |
| --- | --- |
| [PRD](./prd.md) | Menjelaskan masalah, pengguna, alur, dan batas fitur. |
| [Design](./design.md) | Menjelaskan rancangan teknis yang dipilih untuk memenuhi PRD. |
| [Sprint](./sprint.md) | Membagi target menjadi pekerjaan kecil, berurutan, dan dapat diuji. |
| [Panduan AI agent](./ai-agent-guide.md) | Menjaga konteks, aturan, dan hasil kerja AI tetap konsisten. |

## Aturan perubahan

1. Ubah PRD bila kebutuhan pengguna atau aturan bisnis berubah.
2. Ubah design bila pilihan teknologi, model data, atau arsitektur berubah.
3. Ubah sprint sebelum memulai pekerjaan baru atau setelah prioritas berubah.
4. Jangan menyimpan password, URL database asli, token, atau data pelanggan di folder ini.
